Step 7 — Listing Docker Containers

how to see all active and inactive all container 
docker ps

only active container see
docker ps -a

veiw latest container 
docker ps -l

how to stop container force fully remove (-f)
docker stop container-id

how to run container in dettach mode (-d)
docker run -d nginx

how to assign name of the container --name c1
docker run --name -d nginx

how to list all container id 
echo $(docker ps -aq)

we want remove all container all 
docker rm -f $(docker ps -aq)

howm to remove images
docker rmi imageid/imagename

direct run 
docker run --name -d nginx

how to see log 
docker logs -f(live log) c1


Step 8 — Pushing Docker Images to a Docker Repository

docker login -u docker-registry-username

docker push docker-registry-username/docker-image-name
